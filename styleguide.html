Bridgetown Source Style Guide

Welcome to the BTSOURCE.NET style guide. The following style guide gives a detailed rationale and explanation of each component and rule used throughout the site.

Logos & Colors

There are various forms of the logo. The main three color logo with the company name. A simple title only logo using shades only, and the graphical logo only. The goal is to use all three of these in different combinations when promoting or representing Bridgetown Source.

Site Colors:

The color palette for the site is derived from the logo set. Below is a full spectrum of the color palette of btsource.net.

Typographic Elements

btsource.net is a small web site therefore I made a point to keep the typographical element count to a minimum. There are five core text elements used on the site. Four headers .section-heading .section-heading-dark .h1 .h3, paragraph formatting, blockqoute and a code hilighting element. Font-sizes range from .p == 16px to .section-heading == 32px; Furthermore, each core typographical element has a contrasting selector which renders a lighter font color for dark backgrounds which are applied using the .light child selector of the main element.

Layout & Grid

The btource.net grid is simple and focused on vertical symmetry. Pages start with an HTML5 header element which can contain any navigation elements. Moving down the page section elements divide the page into discrete categories. There are two base section styles: Light blue background, and a dark blue background. for each .section-light or .section-dark there are corresponding selectors which serve as headings for each section:

Section specific selectors:




.section-light { background-color: #CCE5F6;
    clear: both;}
.section-dark { background-color: #2C3E50;
    clear: both;



Section heading selectors:

There are three section headers, light, skyblue, and darkblue. The light blue heading selector has minimal use, and might get removed in the future. The following selectors format the text alignment and dimensions. The typographic element .section-heading handles the properties of the actual font face. Vertical flow between text elements is handled by the margin settings set by the typograhpic elements. Sitewide vertical spacing between core elements is 46px;

.section-center {
    background-color: #3498DB;
    width: 100%;
    height: 128px;
    line-height: 128px;
    margin: auto;
}
.section-center-dark {
    background-color: #2C3E50;
    width: 100%;
    height: 128px;
    line-height: 128px;
    margin: auto;
}
.section-center-light {
    background-color: #CCE5F6;
    width: 100%;
    height: 128px;
    line-height: 128px;
    margin: auto;
}
