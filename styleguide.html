Bridgetown Source Style Guide
Typographic Elements:

btsource.net is a small web site therefore I made a point to keep the typographical element count to a minimum. There are five core text elements used on the site. Two headers (h1, h3), paragraph formatting, blockqoute and a code hilighting element.
Furthermore, each core typographical element has a contrasting selector which renders a lighter font color for dark backgrounds which are applied using the .light child selector of the main element.

Layout & Grid

Main Grid & Vertical Symmetry:

The btource.net grid is simple and focused on vertical symmetry. Pages start with a HTML5 header element which contain any navigation elements. Moving down the page section elements divide the page into main parts finally ending in a footer.
